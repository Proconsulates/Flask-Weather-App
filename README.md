# Flask-Weather-App
> A simple Flask weather app using the OpenWeatherMap API.

## Technologies
The technologies used to create this awesome app include:
> Flask Templates
> Flask
> CSS
> HTML
> Vanilla JS
> OpenWeatherMap Weather API
> ip-geolocation
> icanhazip
> Python
